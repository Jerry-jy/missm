9、添加监听器，实现获取全部商品类型功能。

10、实现商品增加功能。

11、实现Ajax商品删除功能。

12、实现Ajax商品批量删除功能。

15、实现按商品名称条件查询并分页功能，翻页使用Ajax技术。

16、实现按商品类型查询并分页功能，翻页使用Ajax技术。
